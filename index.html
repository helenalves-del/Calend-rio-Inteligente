Como configurar

  1. Acesse console.cloud.google.com e crie um novo projeto.
  2. No menu lateral: APIs e Serviços → Biblioteca. Ative Google Calendar API.
  3. Vá em Credenciais → Criar credencial → ID do cliente OAuth. Tipo: Aplicativo da Web.
  4. Em Origens JavaScript autorizadas, adicione a URL onde o app vai rodar (ex: https://meusite.vercel.app ou http://localhost:8080).
  5. Copie o ID do cliente gerado e cole na variável CLIENT_ID no topo do arquivo.
  6. Para usar em equipe: em Tela de consentimento OAuth, adicione os e-mails dos colegas como Usuários de teste (enquanto o app estiver em modo de teste).
  7. Hospede o index.html no Vercel, Netlify ou GitHub Pages — gratuito e sem servidor necessário.
